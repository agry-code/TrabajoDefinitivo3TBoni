Proposito del trabajo

La idea de mi proyecto ha sido un wordle de los Simpson, este juego lo podemos encontrar en la barra de navegación.

Debemos considerar que para cumplir la rubrica he implementado unas cuantas páginas más en el proyecto web.

Este verano me gustaría desarollar más este juego implementando varias cosas en el código y algunas ideas son:

  1. Hacer un juego responsive, es decir, dependiendo del nombre elegido de forma aleatoria. El tablero de juego debe aumentar o disminuir.
  2. Hacer un viewport responsive para todos los dispositivos y crear 3 breakpoints: moviles, tablets y tv
  3. Crear un dominio único para Springfidle y poder compartirlo de forma gratuita con el resto de fans de los Simpsons.
  4. Implementar un score y un ranking
  5. Conseguir un buen SEO para el posicionamiento Web
  6. Tener un codigo ordenado y simple que me permita implementar nuevas ideas de forma sencilla
